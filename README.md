<h2> Completely self made C++, DirectX 11 and Direct2D powered game, work heavily under progress</h2>
<p> Features ready: keyboard, graphics card resource loader, animation(for sprites), scrolling background, player motion (jump, duck, walk) & timer(for fps and updating) </p>
<p> Feel free to use, its in the good ground for further developing, there are many bugs tho. If you are using windows 7 you need to install UWP </p>
